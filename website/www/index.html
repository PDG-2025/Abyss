Fonctionnalités essentielles pour votre sécurité

Abyss offre toutes les mesures, alertes et outils nécessaires pour une plongée en toute sécurité et confort.

Mesures en temps réel

  • Profondeur actuelle et maximale
  • Temps de plongée écoulé
  • Limite de non-décompression (NDL)
  • Vitesse de remontée avec alerte
  • Température de l'eau

Gestion de la sécurité

  • Alertes visuelles et sonores
  • Paliers de décompression
  • Intervalle de surface minimum
  • Temps minimum avant l'avion
  • Gestion multi-gaz (air, nitrox)

Navigation et suivi

  • Compass numérique intégré
  • Enregistrement automatique des données
  • Consultation de l'historique
  • Synchronisation Bluetooth
  • Application mobile dédiée

Votre sécurité, notre priorité

Abyss intègre des systèmes d'alerte avancés pour vous protéger dans toutes les situations de plongée.

Alertes intelligentes

Alertes visuelles et sonores pour vitesse de remontée excessive, dépassement de NDL, profondeur maximale atteinte.

Calculs précis

Algorithmes de décompression avancés pour calculer les paliers nécessaires et les intervalles de surface.

Vol après plongée

Calcul automatique du temps minimum à respecter avant de prendre l'avion après votre plongée.

Application mobile dédiée

Synchronisez vos données de plongée via Bluetooth et consultez votre historique détaillé sur votre smartphone.

Fonctionnalités de l'app

  • Visualisation des profils de plongée
  • Statistiques complètes des plongées
  • Paramétrage des alertes de sécurité
  • Configuration des paramètres de l'ordinateur de plongée

Questions fréquentes

Trouvez des réponses aux questions les plus courantes sur Abyss.

Abyss fonctionne avec une batterie Li-ion 18650 de 3500 mAh offrant 3 à 5 heures d'autonomie en mode plongée continue selon l'utilisation. La batterie est facilement remplaçable.

Le capteur MS5837 haute précision permet de plonger jusqu'à 290 mètres de profondeur (30 bar), couvrant tous les types de plongée loisir et technique avec une précision au centimètre.

Abyss utilise le Bluetooth intégré du Raspberry Pi Zero 2 W pour se connecter automatiquement à l'application React Native. Les données sont ensuite synchronisées dans le cloud via une API sécurisée.
